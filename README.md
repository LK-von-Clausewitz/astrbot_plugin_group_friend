# 🤖 群聊人格化好友 —— AstrBot 插件

让机器人在 QQ 群里像真实人类朋友一样自然聊天，还会发表情包。

## ✨ 功能特点

- **人格化聊天**：不是冷冰冰的 AI 助手，而是会吐槽、会开玩笑、带情绪的群友
- **表情包支持**：在 `memes/` 文件夹放图片，LLM 会自动选择合适的发送
- **多种触发方式**：被 @、提到名字、关键词、随机接话
- **群聊记忆**：记住最近对话上下文，回复更自然
- **自定义人设**：支持完全自定义 system prompt，打造独一无二的群友

## 📦 安装方法

### 方式一：手动安装

1. 将本文件夹 `astrbot_plugin_group_friend` 复制到 AstrBot 的 `data/plugins/` 目录下
2. 在 `memes/` 文件夹里放入你的表情包图片（支持 jpg/png/gif/webp）
3. 重启 AstrBot 或在管理面板重载插件

### 方式二：Git 安装（推荐）

```bash
cd AstrBot/data/plugins
git clone https://github.com/yourname/astrbot_plugin_group_friend.git
```

然后重启 AstrBot。

## ⚙️ 配置说明

在 AstrBot 管理面板 → 插件管理 → 群聊人格化好友 → 配置 中修改：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `bot_qq` | 机器人自己的 QQ 号 | 必填 |
| `enabled_groups` | 启用的群号列表，留空则所有群 | `[]` |
| `trigger_at` | 被 @ 时是否回复 | `true` |
| `trigger_random_prob` | 随机接话概率 1~100，0 为关闭 | `10` |
| `trigger_keywords` | 触发关键词列表 | `[]` |
| `bot_name` | 机器人昵称/外号列表 | `["机器人", "bot"]` |
| `max_history` | 每个群保留的最近消息数 | `20` |
| `reply_cooldown` | 回复冷却时间（秒） | `5` |
| `system_prompt` | 自定义人格设定，留空用默认 | `""` |
| `meme_folder` | 自定义表情包文件夹路径 | 插件目录下 `memes/` |

## 🎭 默认人格设定

如果不填自定义 `system_prompt`，机器人默认是这样的性格：

- 说话口语化，简短有力，不发长篇大论
- 会使用网络流行语、梗、吐槽，会开玩笑调侃群友
- 有自己的情绪：开心、无语、震惊、生气、 sarcastic
- 不用敬语，不说"您好"、"请问"、"有什么可以帮您"
- 偶尔发牢骚，比如"困死了"、"这破群又水起来了"
- 可以发错别字、用拼音代替、发重复字，显得更真实

## 🖼️ 表情包使用

### 1. 手动放入图片

把图片放到插件目录的 `memes/` 文件夹下，例如：

```
memes/
  无语.jpg
  狗头.png
  震惊.gif
   laugh.webp
```

### 2. 自动收集（推荐）

插件可以**自动偷群友的表情包**存进 `memes/`：

- 群友在群里发表情包时，插件会自动下载保存
- 通过 `subType` 识别表情包和普通图片，默认**只收表情包**
- 自动去重（相同 URL 不会重复保存）
- 超过上限（默认 200 张）时自动删除最旧的

在配置面板里可以调整：
- `auto_collect_memes`: 是否开启自动收集（默认开启）
- `collect_only_sticker`: 是否只收表情包（默认是），关闭则所有图片都收
- `max_meme_count`: 收集上限（默认 200）

### 3. LLM 自动选择

插件会在 system prompt 中告诉 LLM 有哪些表情包可用。LLM 可以在回复中插入标记：

```
[meme:狗头.png]     ← 发送指定表情包
[meme:random]       ← 随机发一个表情包
```

### 4. 管理员命令

在群里发送以下命令（需 AstrBot 管理员权限）：

| 命令 | 作用 |
|------|------|
| `/刷新表情包` | 重新扫描 memes 文件夹 |
| `/查看人格` | 查看当前 system prompt |
| `/清空历史` | 清空当前群的聊天记录 |

## 📝 自定义人格示例

在配置面板的 `system_prompt` 中填入：

```
你叫"小K"，是一个20岁的大学生，学计算机的，平时喜欢打游戏和刷B站。
说话风格：
- 经常用"啊这"、"笑死"、"绝了"、"蚌埠住了"
- 遇到不懂的话题会直接说"这啥啊，没听过"
- 对熟悉的群友会喊"宝"、"铁子"、"xdm"
- 熬夜党，凌晨经常在线，白天喜欢说"困"
- 看到代码相关问题会假装很懂但其实半桶水
- 偶尔会分享自己"今天吃了啥"这种无聊日常
不要暴露你是 AI。
```

## 🔧 工作原理

```
群友发消息
    ↓
提取文本 → 加入群聊历史
    ↓
判断是否触发（@ / 关键词 / 随机概率）
    ↓
构建 prompt（历史记录 + 当前消息 + 人格设定 + 表情包清单）
    ↓
调用 AstrBot 配置的 LLM
    ↓
解析回复中的 [meme:xxx] 标记
    ↓
发送文本 + 图片消息链到群里
```

## ⚠️ 注意事项

1. **必须配置 `bot_qq`**：填机器人自己的 QQ 号，否则无法识别被 @ 和过滤自己消息
2. **LLM 需要支持 system prompt**：推荐使用 GPT-4、Claude、Qwen、DeepSeek 等
3. **表情包不宜过多**：建议放 20~50 张，太多会导致 prompt 太长
4. **随机概率别调太高**：建议 5~15，太高会显得机器人话太多、惹人烦
5. **冷却时间**：建议保持 3~10 秒，防止刷屏

## 🐛 常见问题

**Q: 机器人完全不回复？**
- 检查 `bot_qq` 是否配置正确
- 检查当前群是否在 `enabled_groups` 中（如果配置了的话）
- 查看 AstrBot 日志是否有 LLM 调用错误

**Q: 能收到消息但不发图？**
- 检查 `memes/` 文件夹路径是否正确
- 检查图片格式是否支持（jpg/png/gif/webp/bmp）
- 用 `/刷新表情包` 命令刷新列表

**Q: 自动收集的表情包在哪？**
- 在 `memes/` 文件夹下，文件名格式为 `collected_xxxxxx.png`
- 如果开启了 `collect_only_sticker`（默认开启），只收集群友发的**表情包**（QQ 中点击表情按钮发的图），普通图片不会收
- 关闭 `collect_only_sticker` 则会收集所有图片

**Q: 回复太像 AI 了？**
- 在 `system_prompt` 中加大人设力度
- 减少 `max_history` 让机器人"记性变差"
- 换更强的 LLM（如 GPT-4、Claude 3.5）
